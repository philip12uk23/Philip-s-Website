How I Made This Web Page

  1. Open Visual Studio Code (VS Code)
  2. Type <!DOCTYPE html>
  3. Next type <html>
  4. Then type <head>
  5. Add a <title> with your name, then close it
  6. Close the </head> tag
  7. Next type <body>
  8. Then start writing your content

Meaning of Common HTML Tags

  • H — Heading
  • U — Underline
  • B — Bold
  • P — Paragraph
  • Br — Break
  • Li — List Item
  • Ol — Ordered List (numbered)
  • Ul — Unordered List (bullets)
  • Mark — Highlight
  • Strong — Bold (semantic)
